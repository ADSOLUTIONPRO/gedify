Créer un espace Gedify local

Gedify tourne en local sur ce Mac (données dans ton dossier perso), et lit tes documents depuis un Paperless existant (le tien, sur ton réseau ou en ligne). Aucun document n'est dupliqué.

HTTP accepté en réseau local ; HTTPS recommandé pour un serveur distant.

💡 Où le trouver : ouvre Paperless → menu utilisateur (en haut à droite) → Mon profil → champ Token API (bouton copier). Ouvrir Paperless

Installer Gedify Local complète

Tout est automatique. L'app installe Paperless en local (PostgreSQL, Redis, Tika, Gotenberg), crée le compte admin et récupère le token, puis lance Gedify local. Aucun token à saisir.

Seul prérequis : Docker Desktop installé et lancé. Paperless reste accessible sur localhost:8010.
